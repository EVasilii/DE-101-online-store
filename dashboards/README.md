## В данном разделе представлены три дашборда, выполненные в разных BI инструментах.

Интерактивный дашборд в Power BI Desktop.<br>
Состоит из трех страниц, на главной представлены ключевые метрики, динамика изменения продаж и прибыли с течением времени, 
а также разбивка по категориям, сегментам и методам доставки. Добавлены две кнопки, с помощью которых можно перейти на другие
страницы и увидеть более подробную информацию о продуктах по локациям и заказчиках.
